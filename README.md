# PeterH3G shadcn

## Setup with [Bun.sh](`https://bun.sh`)

```bash
# Make sure to install the dependencies:
bun install

# Development Server: http://localhost:3000
bun run dev

# Build the application for production:
bun run build

# Locally preview production build:
bun run preview
```
